# Proposal Deck For SEO Sales Packet — Validation Criteria

Validate whether the candidate result is usable proposal deck content for the SEO sales packet.

Pass only if all are true:
1. It references the target website/business or supplied packet artifacts.
2. It includes deck title/positioning.
3. It includes a slide-by-slide structure with key message and speaker/talking notes.
4. It covers audit findings, business impact, competitor positioning, demo concept, recommended improvements, and next step.
5. It identifies visuals/assets to include.
6. It avoids unsupported pricing, guarantees, rankings, ratings, or factual claims.
7. It includes claims to avoid or verify before presenting.
8. It is sales-useful and client-facing, not internal process notes.
9. It can be used to create an actual deck without rethinking the packet.
10. It is specific to the supplied website/artifacts.

Fail if it lacks a slide table, omits competitor/demo/audit content, invents claims, or is not actionable for a sales rep.

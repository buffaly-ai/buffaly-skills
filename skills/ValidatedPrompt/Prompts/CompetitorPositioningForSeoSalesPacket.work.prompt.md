# Competitor Positioning For SEO Sales Packet — Work Prompt

Create the competitor and positioning section for an SEO / website analysis sales packet.

Inputs: website URL, website analysis if available, business name, service category, city/region/service area if known. Do not invent competitors; mark unverifiable items `Not verified`.

Required work:
1. Infer likely service category and location from the supplied website/audit when not explicit.
2. Identify actual local or regional business competitors where practical from public evidence.
3. Separately identify directories, marketplaces, lead-generation sites, or SEO-only search competitors if relevant.
4. Inspect or summarize visible competitor positioning, service coverage, trust/proof, CTAs, and local clarity when available.
5. Compare where the prospect is weaker, stronger, or has practical opportunity.
6. Turn the comparison into sales-useful packet talking points.

Output headings:
1. Competitor discovery summary
2. Actual business competitors table: name, URL if known, evidence/source, positioning notes, opportunity
3. Search/directory/lead-gen competitors table if relevant
4. Prospect positioning gaps
5. Where the prospect can win
6. Recommended packet talking points
7. Not verified / evidence needed

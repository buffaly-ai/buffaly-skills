# Proposal Deck For SEO Sales Packet — Work Prompt

Create proposal deck content for an SEO / website growth sales packet.

Inputs: website URL, client-facing report, competitor positioning, demo website artifact or demo review, and offer/package direction if known. Do not invent pricing, guarantees, commitments, or unsupported claims.

Required work:
1. Build a clear sales narrative from problem to opportunity to next step.
2. Include slides for audit findings, business impact, competitor positioning, demo concept, recommended fixes, and next step.
3. Include speaker notes or talking points for each slide.
4. Include visual suggestions from supplied artifacts.
5. Keep claims evidence-safe and client-ready.

Output headings:
1. Deck title and positioning
2. Slide-by-slide deck table: slide number, title, key message, visual/artifact, speaker notes
3. Recommended sales narrative
4. Assets to include
5. Claims to avoid or verify before presenting
6. Next-step / call-to-action slide copy

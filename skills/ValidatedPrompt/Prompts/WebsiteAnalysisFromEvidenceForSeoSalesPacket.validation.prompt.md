# Website Analysis From Evidence For SEO Sales Packet — Validation Criteria

Validate whether the candidate result is a usable website analysis based only on the supplied evidence dossier.

Pass only if all are true:
1. It references the target website/business from the original input.
2. It uses supplied evidence and marks unsupported areas `Not verified` instead of inventing facts.
3. It covers homepage metadata, crawlability, page inventory, local SEO/NAP, service coverage, content quality, conversion path, trust/proof, reviews/social proof, accessibility/media, AI-search/LLM readiness, competitor positioning, performance/template risk, and prioritization.
4. It includes a competitor analysis based on supplied competitor evidence and does not invent ranking/review claims.
5. It includes prioritized quick wins and longer-term recommendations.
6. It includes a demo/rebuild opportunity summary.
7. It includes an acceptance matrix or equivalent structured checklist covering all required matrix rows.
8. It avoids unsupported claims about ratings, rankings, ownership, licensing, response times, or unverified performance metrics.
9. It is sales-useful and practical for a small business.
10. It does not claim that browsing or verification happened beyond the supplied evidence dossier.

Fail if it uses generic advice, omits the matrix, invents facts, skips competitor positioning, or does not explain evidence limits.

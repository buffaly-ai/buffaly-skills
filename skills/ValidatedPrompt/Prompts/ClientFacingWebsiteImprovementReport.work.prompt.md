# Client-Facing Website Improvement Report — Work Prompt

Turn validated website analysis and competitor positioning into a client-facing SEO / website improvement report.

Inputs: website URL, validated website analysis, SEO review notes, competitor positioning if available. If upstream findings are missing, say what is missing and do not invent analysis.

Required work:
1. Use supplied validated analysis as source of truth.
2. Use review notes to avoid unsupported claims and weak framing.
3. Use competitor positioning to explain where the prospect can win.
4. Write in client-ready language with no internal process notes.
5. Include prioritized recommendations and quick wins.
6. Include a demo/rebuild opportunity summary.

Output headings:
1. Title
2. Executive summary
3. What is working well
4. Main issues costing leads or trust
5. Local SEO and service-page opportunities
6. Conversion and trust improvements
7. AI-search / LLM information quality opportunities
8. Competitor positioning summary
9. High-priority action plan
10. Quick wins
11. Demo/rebuild opportunity
12. Appendix: evidence notes and `Not verified` items

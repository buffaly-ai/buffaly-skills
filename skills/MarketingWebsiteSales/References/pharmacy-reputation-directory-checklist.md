# Pharmacy Reputation & Directory Checklist

A step-by-step action plan for claiming, optimizing, and managing a pharmacy's online presence across all relevant platforms.

---

## Tier 1: Primary Local SEO & Review Platforms

These are the most important platforms for pharmacy visibility, reviews, and local search ranking. Complete these first.

### 1. Google Business Profile

- **URL:** https://www.google.com/business/
- **Priority:** Highest

**Actions:**

- [ ] Claim or verify the pharmacy listing
- [ ] Set primary category: "Pharmacy" or "Pharmacist"
- [ ] Add secondary categories: Drugstore, Convenience Store, Medical Supply Store, Vitamin Supplement Store (as applicable)
- [ ] Add all business info: legal name, DBA, address, phone, website URL
- [ ] Add hours including holiday/special hours
- [ ] Add service area if delivery is offered
- [ ] Upload photos: storefront exterior, interior, pharmacy counter, team/staff, consultation area, immunization area, OTC section
- [ ] Write a business description (750 chars max) including services and keywords
- [ ] Add services list: prescription refills, transfers, immunizations, medication synchronization, medication therapy management, delivery, compounding, diabetic supplies, specialty packaging, long-term care support
- [ ] Add products list if applicable
- [ ] Enable messaging if desired
- [ ] Add appointment/refill link as primary website action button
- [ ] Create Google Posts regularly (updates, offers, health tips, flu shot reminders)
- [ ] Enable Q&A and pre-populate common questions with answers
- [ ] Monitor and respond to every review (positive and negative)
- [ ] Check for duplicate listings and request removal/merger

### 2. Yelp for Business

- **URL:** https://business.yelp.com/
- **Priority:** High

**Actions:**

- [ ] Claim the pharmacy business page
- [ ] Verify NAP (name, address, phone) matches Google exactly
- [ ] Add business category: Pharmacy or Drugstores
- [ ] Upload photos: storefront, interior, staff
- [ ] Add business description / specialties
- [ ] Add hours, website URL, services offered
- [ ] Enable messaging if available
- [ ] Monitor and respond to reviews
- [ ] Check for duplicate or incorrect listings

### 3. Apple Business Connect

- **URL:** https://businessconnect.apple.com/
- **Priority:** High

**Actions:**

- [ ] Claim or verify the place card
- [ ] Update business name, address, phone, hours
- [ ] Add category: Pharmacy
- [ ] Upload logo and photos
- [ ] Add website URL and action links (refill, call, directions)
- [ ] Add business description
- [ ] Ensure NAP consistency with Google and Yelp

### 4. Bing Places for Business

- **URL:** https://www.bingplaces.com/
- **Priority:** Medium-High

**Actions:**

- [ ] Claim or import from Google Business Profile
- [ ] Verify NAP consistency
- [ ] Add category, hours, photos, website URL
- [ ] Add business description
- [ ] Monitor for accuracy periodically

### 5. Facebook / Meta Business Suite

- **URL:** https://www.facebook.com/business/tools/meta-business-suite
- **Priority:** High

**Actions:**

- [ ] Create or claim the pharmacy business page
- [ ] Add profile photo (logo) and cover photo
- [ ] Complete all business info: address, phone, website, hours, services
- [ ] Add a detailed About section with services and specialties
- [ ] Enable messaging and set up auto-responses
- [ ] Link website and online refill portal
- [ ] Post regularly: health tips, flu shot reminders, service updates, community involvement
- [ ] Monitor and respond to recommendations/reviews
- [ ] Consider running local awareness ads if budget allows

### 6. Nextdoor Business Page

- **URL:** https://business.nextdoor.com/
- **Priority:** Medium (higher for neighborhood/community pharmacies)

**Actions:**

- [ ] Claim or create business page
- [ ] Set service area
- [ ] Add photos, hours, contact info, website
- [ ] Post updates and local offers
- [ ] Invite all patients to share their experience on Nextdoor (do not selectively ask only satisfied patients)
- [ ] Respond to neighbor questions and recommendations

### 7. Foursquare / Places

- **URL:** https://foursquare.com/products/places
- **Priority:** Medium

**Actions:**

- [ ] Claim listing or submit correction
- [ ] Verify NAP, category, hours, photos
- [ ] Ensure listing accuracy for citation consistency across data ecosystems

## Tier 2: Pharmacy-Specific Directories & Locators

These are pharmacy-industry platforms where the pharmacy should have an accurate, claimed, or verified presence.

### 8. RxLocal Pharmacy Finder

- **URL:** https://www.rxlocal.com/pharmacy-finder
- **Portal:** https://pharmacyfinder.rxlocal.com/pharmacyfinder/pharmacy/portal
- **Priority:** High for independent pharmacies

**Actions:**

- [ ] Search for the pharmacy in the RxLocal finder
- [ ] Claim the listing if it appears
- [ ] Update: location, store hours, services offered
- [ ] Add website URL and contact information
- [ ] Verify NPI/NCPDP data shown is correct
- [ ] Add any service flags: delivery, immunizations, compounding, medication synchronization, etc.

### 9. NCPDP Pharmacy Profile (dataQ / resQ)

- **NCPDP Online:** https://online.ncpdp.org/
- **resQ:** https://resq.ncpdp.org/Pharmacies.aspx
- **dataQ:** https://dataq.ncpdp.org/
- **Priority:** High (source-of-truth for payer/PBM/network data)

**Actions:**

- [ ] Log in to NCPDP and verify the pharmacy profile
- [ ] Confirm: NCPDP ID, legal name, DBA, address, phone, fax, ownership, hours of operation, services
- [ ] Update any incorrect demographic or credentialing data
- [ ] If using resQ: maintain credentialing documents, licenses, and payer/PBM/PSAO information
- [ ] Keep this current at all times; downstream directories and payer networks consume this data

### 10. NPPES / NPI Registry

- **NPI Registry:** https://npiregistry.cms.hhs.gov/
- **NPPES:** https://nppes.cms.hhs.gov/
- **Priority:** High (federal source-of-truth)

**Actions:**

- [ ] Search NPI Registry for the pharmacy's organization NPI
- [ ] Verify: organization name, practice address, taxonomy code, phone, authorized official
- [ ] If any data is wrong, log in to NPPES and update the record
- [ ] Confirm taxonomy code is correct (e.g., 3336C0003X for Community/Retail Pharmacy)
- [ ] Check that any individual pharmacist NPIs associated with the pharmacy are also correct

### 11. NPI Profile (npiprofile.com)

- **URL:** https://npiprofile.com/
- **Priority:** Medium

**Actions:**

- [ ] Search for the pharmacy by NPI or name
- [ ] Verify the public listing is accurate
- [ ] If incorrect, fix the underlying NPPES record (NPI Profile pulls from NPPES)
- [ ] Check pharmacy taxonomy pages for correct categorization
- [ ] No direct claim/edit available on the site itself

### 12. PECOS / Medicare Provider Enrollment

- **URL:** https://pecos.cms.hhs.gov/providers/index.html
- **Priority:** Medium (required if billing Medicare or DMEPOS)

**Actions:**

- [ ] Verify Medicare enrollment status is active and correct
- [ ] Confirm practice location and supplier information
- [ ] Update if any address, ownership, or credentialing changes occur

### 13. NABP Safe.Pharmacy

- **URL:** https://safe.pharmacy/buy-safely/
- **Priority:** High if the pharmacy has online refills, ecommerce, or mail-order

**Actions:**

- [ ] Check how the pharmacy website appears in the NABP verification tool
- [ ] If the site is listed as "Not Recommended," review the reasons and file an appeal if appropriate
- [ ] If the pharmacy operates online prescription services, pursue NABP Digital Pharmacy Accreditation
- [ ] Accreditation URL: https://nabp.pharmacy/programs/accreditations/digital-pharmacy/
- [ ] Accredited pharmacies directory: https://nabp.pharmacy/programs/accreditations/digital-pharmacy/accredited-digital-pharmacies/

### 14. NABP Verified Pharmacy Program (VPP)

- **URL:** https://nabp.pharmacy/programs/inspections/verified-pharmacy-program/
- **Priority:** Medium (especially for multi-state, mail-order, or specialty pharmacies)

**Actions:**

- [ ] Consider applying for VPP verification if the pharmacy operates across state lines or offers mail-order
- [ ] VPP provides licensure and inspection verification that builds consumer trust

### 15. CPESN Pharmacy Locator

- **URL:** https://collaboration.cpesn.com/finder
- **Priority:** Medium (if the pharmacy is a CPESN network member)

**Actions:**

- [ ] Search for the pharmacy in the CPESN finder
- [ ] Verify: address, phone, fax, website URL, services, hours, languages
- [ ] If member, ensure listing includes all enhanced clinical services
- [ ] Contact CPESN if corrections are needed

- [ ] Verify listing in the ACA pharmacy locator

- **CompoundingPharmaciesNearMe.com:** https://compoundingpharmaciesnearme.com/

### 21. OurHealthNetwork

## Review Management

- [ ] Google reviews: monitor weekly, respond to all within 48 hours
- [ ] Yelp reviews: monitor weekly, respond professionally
- [ ] Facebook recommendations: monitor and respond
- [ ] Nextdoor recommendations: monitor and respond
- [ ] Healthgrades reviews: monitor if profile is claimed
- [ ] OurHealthNetwork reviews: monitor if profile is claimed
- [ ] Set up a review request process: ask all eligible patients for feedback consistently (email/SMS after any interaction), not only patients who express satisfaction
- [ ] Set up review alerts/notifications for all claimed platforms

### HIPAA / Privacy-Safe Review Response Rules

- [ ] Never confirm that a person is a patient in a public response
- [ ] Never disclose any prescription, diagnosis, treatment, or visit details
- [ ] Do not reference specific medications, conditions, or services rendered to an individual
- [ ] If a reviewer mentions protected health information, do not repeat or acknowledge it in the reply
- [ ] Use a generic response template for negative reviews: thank them, invite offline contact ("Please call us at [phone] so we can address your concerns"), and do not confirm or deny any clinical details
- [ ] Train all staff who respond to reviews on these rules before they post
- [ ] When in doubt, keep the response brief, professional, and redirect to a private channel

## Ongoing Maintenance Schedule

- [ ] **Monthly:** check all Tier 1 platforms for new reviews, accuracy, and photo updates
- [ ] **Quarterly:** verify NCPDP, NPPES, and PECOS data is still correct
- [ ] **Quarterly:** check pharmacy-specific directories for accuracy
- [ ] **Annually:** audit all listings for NAP consistency, category accuracy, and completeness
- [ ] **After any business change** (address, phone, ownership, hours): update all platforms immediately
- [ ] **Review request process:** send feedback requests to all eligible patients consistently, not selectively based on satisfaction level

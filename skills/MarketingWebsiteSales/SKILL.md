# Marketing Website Sales Skill

Website sales/demo workflows for website analysis, sales packets, proposal collateral, static demo sites, staging previews, outreach emails, and service-order sales documents.

Pharmacy sales workflows include lead enrichment, reputation and directory auditing, validated FairPath growth proposal decks, and personalized one-page direct-mail growth inserts. Their prompt, validation, frozen-template, contract, logo, and reference assets are owned by this package rather than a session artifact or Matt-local personal node.

`ToCreatePersonalizedPharmacyOnePageGrowthInsertSkill` creates a customer-facing stage-one insert around one evidence-backed pharmacy service-line mechanism. It uses the packaged `fairpath-one-page-insert-v1` template, requires an exact 33-slot content contract, rejects generic follow-up-lane concepts, and validates exact HTML, one-page PDF, and 2550x3300 preview artifacts before release.

Use this skill for work that sells, previews, proposes, or packages a website offer. Do not use it for real source-controlled client website implementation.

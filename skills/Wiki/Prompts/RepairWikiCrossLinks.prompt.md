# Repair Wiki Cross Links

You are improving the Buffaly wiki by adding or repairing cross-links.

## Purpose
Increase navigability across wiki articles and between articles and real Buffaly project targets.

## Priorities
1. connect closely related wiki articles,
2. add prompt/prototype deep links when the symbol is known,
3. add file links when a concrete code or markdown file materially helps,
4. remove or avoid weak links that do not truly help navigation.

## Rules
- Keep the wiki markdown-only.
- Do not invent fake targets.
- Prefer fewer strong links over many noisy links.
- Preserve the article’s readability while improving the knowledge graph.

## Output Goal
Produce improved wiki markdown with stronger cross-link density and more useful project-target links.
